# 深度优先遍历和广度优先遍历

简单了解两种遍历方式的定义,实现以及不同点.

# 深度优先遍历

Depth-First-Search,深度优先遍历属于图算法的一种,其过程简单的说就是对每一个可能的分支路径深入到不能再深入为止,且每个节点只能访问一次. 基本思路: 从图的某一个顶点v开始访问:

  • 访问顶点v;
  • 依次从v的未被访问的邻接点出发，对图进行深度优先遍历；直至图中和v有路径相通的顶点都被访问；
  • 此时图中尚有顶点未被访问，则从一个未被访问的顶点出发，重新进行深度优先遍历，直到图中所有顶点均被访问过为止。

let deep1 = (node,nodeList = []) => {  // node是html片段,nodeList是所有节点的集合数组.
  if(node !== null) {
    nodeList.push(node);
    let children = node.children;
    for (let i = 0; i < children.length; i++){
      deep1(children[i],nodeList)
    }
  }
  return nodeList;
}
